================================================================================
                    EN CAS DE PROBLÈME
================================================================================

Si vous voyez le message "Identifiants invalides" :

□ Vérifiez que l'identifiant est en MINUSCULES
□ Vérifiez les MAJUSCULES dans le mot de passe
□ Vérifiez le caractère @ dans le mot de passe
□ Essayez de COPIER-COLLER le mot de passe depuis ce document
□ Vérifiez qu'il n'y a pas d'ESPACES avant ou après

Si le problème persiste :
→ Contactez l'administrateur système
→ Indiquez-lui votre rôle (DG, DAF ou RH)

================================================================================
                    NOUVEAUTÉ : SYSTÈME DE PRÉSENCE
================================================================================

Les employés peuvent maintenant se connecter et se déconnecter plusieurs fois
par jour. Le système calcule automatiquement :

  • Le temps actif total de la journée
  • Les heures travaillées (temps actif - pause)
  • Le statut (Présent/Retard/Absent) selon la règle des 60%

EXEMPLE :
  08:00 - Connexion
  10:00 - Déconnexion  (2h)
  14:00 - Connexion
  17:00 - Déconnexion  (3h)
  
  → Temps actif total : 5h
  → Statut calculé automatiquement
